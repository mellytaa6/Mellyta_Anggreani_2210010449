Praktek Pemrograman Web

Sesi 1

Praktek Pemrograman web 1 akan membahas tentang cara menampilkan halaman web dengan menggunakan HTML, CSS, dan Javascript. Mahasiswa diharapkan mampu memahami sintaks dasar dalam menampilkan halaman web, dan dengan pemahaman tersebut menuliskan hasil kreasisnya sendiri.

  • HTML
  • CSS
  • Javascript

Pada sesi 1 mahasiswa mengenali dulu struktur dasar HTML beserta sintaks dasarnya.

  1. Pengenalan HTML
  2. Tabel
  3. Form
